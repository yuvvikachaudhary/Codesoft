# Todo Application in Python

## Description

Todo Application is a command-line based task manager built using Python. It allows users to add, edit, and delete tasks, as well as mark them as completed. The app is designed to be simple, lightweight, and easy to use.

## Features

- Add tasks: Easily add new tasks to your todo list.
- Edit tasks: Update task details .
- Delete tasks: Remove tasks from your todo list when they are no longer needed.
- List tasks: Display all tasks.
- Save data: Tasks are saved in a local JSON file, ensuring data persistence across sessions.

## How to Use

1. Make sure you have Python 3.x installed on your system.
